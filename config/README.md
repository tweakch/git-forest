# Configuration

This directory contains configuration files for git-forest.

## Files

- `default.json` - Default configuration settings
- `forest.schema.json` - JSON schema for forest configuration

## Configuration Options

### Forest Settings
- `forest.name` - Name of the forest
- `forest.defaultPath` - Default path for the forest
- `forest.autoDiscovery` - Automatically discover git repositories

### Git Settings
- `git.defaultBranch` - Default branch name
- `git.autoFetch` - Automatically fetch updates
