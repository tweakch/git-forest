# status

Show the status of the current forest.

## Usage

```bash
git-forest status
```

## Description

The `status` command displays information about the current forest, including:
- Number of plants (repositories)
- Current planters
- Active planners
- Forest health status

## Example

```bash
$ git-forest status
Forest Status:
Name: my-forest
Plants: 3
Planters: 2
Planners: 1
```
